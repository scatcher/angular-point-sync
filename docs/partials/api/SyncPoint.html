SyncPoint

Description

Constructor to handle notifying models when data is updated.

Dependencies

  • apConfig
  • apConfig#firebaseURL

Usage

SyncPoint(model, updateQuery);

Parameters

ParamTypeDetails
modelobject

Model to notify for change events.

updateQueryfunction

Query to execute on change.

Returns

object

sync Sync object

Methods

  • registerChange()

    Notify all other users listening to this model that a change has been made.

  • subscribeToChanges(callback)

    Allows subscribers (controllers) to be notified when change is made

    Parameters
    ParamTypeDetails
    callbackfunction

    Callback to execute after a change is made.
